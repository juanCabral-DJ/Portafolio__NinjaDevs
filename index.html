Sobre Mi

Soy un Desarrollador de Software especializado en la transformación de procesos manuales y datos dispersos en ecosistemas digitales eficientes. Mi enfoque técnico combina el diseño de arquitecturas Full Stack y la implementación de flujos ETL robustos, utilizando tecnologías como .NET y Java para cimentar el backend, mientras aprovecho la potencia de Power BI para traducir información compleja en dashboards estratégicos.

Para asegurar la calidad y el éxito de cada entrega, baso mi gestión de proyectos en metodologías ágiles, orquestando el ciclo de vida del desarrollo mediante Scrum. Utilizo herramientas líderes como Jira y Azure DevOps para mantener la alineación con los objetivos del negocio. Asimismo, aplico estándares rigurosos de control de versiones con Git y Git Flow, fomentando un entorno colaborativo y organizado que permite una ejecución técnica impecable.

Mis Proyectos Destacados

Soluciones a problemas reales utilizando automatización, analisis de datos y desarrollo web

Sistema de Facturacion

Problema: La empresa necesitaba migrar su proceso de facturación de forma manuscrita a un sistema de facturacion que le permita llevar un control del inventario, generar reportes en Excel sobre los clientes, productos y poder generar las facturas y guardarlas como documentos PDF.

Solución: Se creo un sistema de escritorio funcional utilizando Java y MVC en el cual se le permitia a la empresa registrar sus clientes, inventario y facturas, para asi poder llevar un control de esto. Se le agregaron las funcionalidades de poder hacer reportes de los clientes y productos en excel y que al momento de generar una factura esta se imprima y se guarde como un documento PDF en el servidor local.

Pruebas Automatizadas con Selenium

Problema: La validación manual de funcionalidades críticas en las aplicaciones web (como el inicio de sesión y la gestión de datos) consumía una cantidad excesiva de tiempo repetitivo y estaba propensa a errores humanos, lo que retrasaba los ciclos de despliegue y aumentaba el riesgo de fallos en producción.

Solución: Se creó una suite de pruebas automatizadas utilizando C# y Selenium WebDriver. El sistema simula interacciones reales de usuario para validar flujos complejos (como el login y creación de repositorios) de forma desatendida, asegurando la calidad del software, reduciendo drásticamente el tiempo de regresión y garantizando la estabilidad de las funcionalidades clave.

Tienda Online de Ropa

Problema: La falta de un canal de ventas digital limitaba el alcance del negocio estrictamente al mercado local. Además, la gestión del inventario manual no permitía visualizar el stock disponible en tiempo real, dificultando la planificación de reabastecimiento.

Solución: Se está construyendo una arquitectura robusta de E-commerce diseñada para la escalabilidad. La solución integra un catálogo dinámico de productos, una gestión lógica del carrito de compras y autenticación segura, enfocada en sincronizar el inventario con las órdenes generadas.

Modulo de Registro de Usuarios

Problema: La plataforma carecía de un proceso de incorporación de usuarios estandarizado y seguro. La gestión ineficiente de las nuevas cuentas generaba datos inconsistentes y vulnerabilidades de seguridad, lo que impedía una segmentación adecuada de los clientes y comprometía la integridad de la base de datos.

Solución: Se desarrolló el módulo de registro centrado en la seguridad y la integridad de datos. La solución implementa validaciones robustas tanto en el front-end como en el back-end, utiliza encriptación de credenciales y establece una arquitectura escalable que garantiza un punto de entrada fiable para la gestión de clientes.
